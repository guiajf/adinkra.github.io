# Oráculo Adinkra - sabedoria ancestral

Um aplicativo web progressivo (PWA) que funciona como oráculo de sabedoria ancestral, 
utilizando os símbolos Adinkra da cultura Akan de Gana.

## Sobre os Símbolos Adinkra

Os símbolos Adinkra são formas visuais com significado histórico e filosófico, 
originárias do povo Gyaman de Gana e Costa do Marfim. Saturados de significado, 
estes símbolos representam a riqueza da cultura Akan e servem como uma forma 
visual de comunicar verdades profundas.

Fonte: [Adinkra Symbols & Meanings]

## Desenvolvimento

**Código gerado por Inteligência Artificial** (Kimi 2.5) com base em 
especificações do usuário e pesquisa sobre simbolismo Adinkra.

- Estrutura HTML/CSS/JavaScript
- Representações SVG dos símbolos
- Design responsivo para dispositivos móveis

## Como Usar

1. Abra o site em um navegador mobile
2. Toque na tela para revelar um símbolo aleatório
3. Leia a sabedoria ancestral associada
4. Toque "Nova Consulta" para retornar

## Instalação (PWA)

No Android: Menu → Adicionar à Tela Inicial
